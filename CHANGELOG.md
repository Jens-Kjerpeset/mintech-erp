# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-beta.2] - 2026-03-22
### Added
- Interactive invoice PDF generation layer (mostly client-side).
- Deep offline capabilities via Dexie.js for fair/trade show durability.
### Fixed
- Huge memory leak when generating multiple large invoices in a row.

## [1.0.0-beta.1] - 2026-02-14
### Added
- Initial brutalist design system integration.
- Zustand store wiring for dynamic catalog rendering.
### Changed
- Replaced light mode tokens with absolute black/white contrasting blocks for high visibility environments.
